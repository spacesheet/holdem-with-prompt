프레임 및 컴포넌트 구성

1. 입력은 체크 박스(Y/N)로 구성
- 프레임 여부
Y: 창 프레임(닫기, 최소화 등), 기본값 화면 90%
           N: 컴포넌트 (위젯화, 기본값 네모 여부)
            ㄴ 컴포넌트 네모X = 합성 함수 + 음함수 입력
- horizontal numbers
- vertical numbers

2. size 조절 = 드래그해서 너비, 높이 늘림 (기준 수치 기반 Cache)

3. 컴포넌트의 경우 = 드래그해서 프레임에 종속되게 (기준점 값 기반 CRUD)
